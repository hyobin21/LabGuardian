# 🛡️ LabGuardian™: AI-Driven Intelligent Lab Safety Control System

**[인공지능프로그래밍] 학기 프로젝트 및 아이디어톤 출품작**

## 📝 프로젝트 소개
**LabGuardian™**은 연구실 내 미세한 가스 유출 및 독성 물질 노출을 실시간으로 감지하고, AI가 상황 맥락을 분석하여 관리자에게 즉각적인 대응 정보를 제공하는 **이중 하이브리드 AI 기반 안전 관제 시스템**입니다.

- **대상**: 대학원 및 소규모 연구실
- **차별성**: 기존의 단순 임계치 감지 방식을 넘어, 128채널의 다차원 센서 데이터를 정밀 분석하고 시계열 맥락(야간/지속성)을 통합 판단함으로써 연구실의 안전 사각지대를 완전히 제거합니다. 
이미 사고가 일어난 후에 울리는 기존 경보기들과 달리 실시간 모니터링 및 소량 장기 누출도 파악하여 연구자의 **만성 노출을 예방**하는 시스템입니다.

## ai 리포트 기능 (왼쪽 메뉴)
- 또한 일일리포트 기능과 주간 리포트 기능이 있어 안전관리자가 상주하지 않는 랩실에서 자체적으로 ai가 만든 하루동안의 보고서를 받을 수도 있고, 주간별 연구실 상태 종합점수와 그래프 및 위험상황이 몇 번 일어났는지 확인도 가능하기에 연구실 상태를 점검하고 사고를 예방하는데 도움을 줄 수 있습니다.
- 특히 일일리포트에는 하루동안 무슨 위험 상황이 일어났는지는 물론, 그 위험 상황의 원인과 예방법까지 추론해 도출해주기에 연구실 안전관리, 특히 **예방**에 특화되어 있습니다.
- 일일리포트는 원래 1시간 단위로 연구실의 상태를 기록하지만, 현재 내가 조절 바로 조절한 상태의 등급과 원인, 예방법을 보고 싶을 땐 조절바를 조절하고 일일리포트 기능에 있는 **[현재 시각 데이터 일일 리포트에 반영하기]** 버튼을 사용해 실시간으로 조절한 상태도 기록할 수 있게 만들었습니다.
- 일일리포트는 한국의 현재 시각과 동기화되어 있습니다.

## 왼쪽 실시간 전체 농도 조절 바 (전자코 센서 역할)
- 이 웹페이지 및 인공지능은 전자코 센서를 달면 바로 작동하게 만들어 놓았으나, 현재 진짜 전자코가 없기 때문에 사용자가 수동 조절을 할 수 있게 조절 바를 달아놓았습니다. 
- 현재는 (각각의 농도는 알려주지 않고) 전체 농도만 조절하면 각각의 가스 농도를 ai가 추론하는 시스템으로 작동합니다.

## 🚀 주요 기능
- **AI 1 (Sensory Intelligence):** 디리클레 분포 기반 합성 데이터를 활용한 6종 가스 농도 정밀 추론
- **AI 2 (Contextual Intelligence):** 시간대별 위험 맥락 및 노출 지속시간을 반영한 5대 위험 시나리오 판정
- **Real-time Dashboard:** Streamlit 기반의 엔터프라이즈급 관제 콘솔 제공
- **Emergency Interlock:** 위험 등급(Class 0-4)에 따른 동적 UI 경보 및 비상 대응 시스템
